# 黄栗子『私人订制』——个人内核篇：8-『情商』第8节：人际交往中的读心术

🎼先在看一个案例，李四是张三的客户，张三希望约他吃一次饭，在饭桌上谈下一单生意。张三对李四说道，周六我们吃个饭吧。李四回答，不好意思，周六没时间。张三说，再忙也要吃饭啊，吃晚饭总可以吧。李四回答。

不好意思，真的没时间。张三说，那能告诉我，你哪天有时间吗？李四回答，不知道，再来看看用了本期说话技巧后的效果。张三对李四说，明天周六了，我们一起吃饭还是看电影。李四回答道，不好意思，没有时间。

张三继续说，那就吃个便饭吧，反正你一个人吃也是吃，两个人吃也是吃。李四回答，吃个饭倒也行，张三说，那就这么设定呢？为什么两种沟通方式带来的效果会是天壤之别呢？例子先给大家卖个关子，这个说话技巧啊。

也只是本期课程所讲技巧中的一个而已。我将这些说话技巧统称为人际交往中的读行术，其本质上是应用了心理学中的巴拉姆效应。🎼他能在短时间内引起对方共鸣，让对方感觉自己被理解被认同，从而快速建立起彼此的信任感。

巴拉姆效应具体指的是人们常常会认为一些笼统的一般性的人格描述准确的揭示了自己的特点。所以当人们用一些含糊不精，广泛的形容词来描述另一个人的时候，他往往就更容易接受这些描述，并认为描述中所说的就是自己。

例如，你表面看起来很花心。其实我知道你很专情，我觉得你是一个很有想法的人，但好像很多人都不理解你等等这样的话，描述其实非常笼统。但一旦别人对你说了这些话，你就会下意次觉得说的对，说的准。

那为什么这样笼统含糊的描述会让人觉得说的就是自己呢？这是因为人最关心的永远是自己。因此当你在和别人沟通时，只要说出的话趋向于对方的需求，他就会在潜意识里选择去相信。那么如此一来，信任感就快速建立起来了。

信任是人际关系的基石，如果缺乏信任。无论是和。🎼家人还是朋友亦或是同事，都很难会有顺畅的互动。这也是为什么当代人都特别希望别人能够了解我，信任我，接纳我，其本质上啊就是为了获得认可。

而读行术的沟通就是基于这个目的。举个例子，张三对李四说，你最近发生什么呢？李是反问干嘛？张三说，你最近对人很冷漠啊。李是回答，那又怎样？张琛继续说，以前你性格挺好的啊，现在咋变了？李是回答。

这关你什么事儿。🎼这场对话充满浓浓的火药味，明明张三没有恶意，但李斯却觉得张三多管闲事，导致张三碰了一鼻子灰，再来看看能够获得认同感的沟通。张三对李四说，你最近给人感觉很冷漠啊。李四回答，那又怎样？

张三说，上次看见你和你男朋友在一起的时候，特别温柔。你对家人应该是很温柔的吧。李四回答，是啊，对在乎的人当然温柔，张三说，果然不出我所料，你其实是一个很温柔的人。李四回答，不好意思啊，刚才对你态度不好。

当你先认同对方，对方就会给你积极的回应。这样信任感自然而然就有了。如果采用最开始的沟通方式去抱怨，指责对方显然会把两个人的关系闹得不愉快。建立在信任之上的人脉是一种宝贵资源。他可以给生活。

给事业创造意想不到的惊喜和价值。因此啊，在人际交往中，懂得如何与他人建立信任感非常重要。下面例子就来给大家详细介绍4种沟通毒性技巧。帮你建立良好的人脉。🎼第一种让对方无法说漏。当我们接到推销电话时。

很少会耐心听完对方的话，通常是接都不会接。即便是接听后，也只想赶紧挂掉。也就是说，当一个人的脑海中已经决定说漏，那要再让他回心转意就很困难了。回顾开头给大家讲的张三约李四吃饭的案例。

第一种情况被无情拒绝。第二种情况被接受，为什么答案很简单？第二个情况中，张三给了李四两个选择，明天周六了，我们一起吃饭还是看电影。🎼这句话就很难让李是说漏，因为人在面临选择时，会下意识思考做出哪种选择。

对自己更有利。所以李斯心里很狠的在想，算了，就和张生吃个饭吧，反正自己也要吃饭。🎼让对方无法说漏这个技巧的奥妙就在于给对方出选择题，让对方二选一，对方会选择有利于自己的那一个选项。🎼再来看一个案例。

小陈和朋友去吃饭，饭店老板问道，要几瓶啤酒，小陈回答道，6瓶吧，1人一瓶。如果用让对方无法松漏这个技巧会产生什么样的效果呢？来看看小曾和朋友去吃饭。饭店老板问道，先给你们上一箱啤酒还是两箱啊。小陈回答。

一箱吧，不够在家。🎼聪明的饭店老板，如果用这个技巧去和顾客沟通，那饭店生意肯定会更加火爆。🎼第二种故意说错话。这个技巧是在对话中故意说错话，引导对方来做修正，借此来获取自己想要得到的信息。

达到自己真正的目的。举个例子，小陈有个项目需要行业内的一位前辈指点，但不好意思直接问前辈具体住在几楼，于是小陈问街坊邻居，张总是住在406吧，知道具体位置的邻居啊，会马上纠正错误。哎。

不是406是409，其实像这样的方式，在生活中，我们经常会用到，只要给出错误的猜想，就可以跨过直接询问，去利用对方想纠错的心理，毫无戒备的获取到自己想要的信息。第三种语言同调，人都有防备之心。

要想悄无声息的打开对方的心扉，就可以用到语言同调这个沟通技巧。所谓同调呢就是在和对方的谈话中，捕捉对方说的话里面的关键词，然后模仿使用这个关键词。举个例子，张三找到了新工作，对李师说，我找到新工作了。

🎼接下来我要加油工作，不断挑战自己。李四回答道，好好干。张三说，你都不鼓励我一下，太敷衍了吧。李四回答，祝你成功。张三说，哎，算了算了，没劲。🎼像李四这样去沟通，不仅人际关系不能更进一步。

而且还容易引起对方的反感。但患者语言同调就完全不一样了。来看一下，张三对李四说，我找到新工作了，接下来我要加油工作，不断挑战自己。李四回答道，真不错，为了给你加油鼓今。尽管我也要挑战一下。

为你做一顿丰盛的晚餐。张三说，谢谢你，你太有心了。你是回答客气啥，看你这么有干劲，我今后也要多做出一些挑战了，为我加油哦。张三说，哈你太有意思，可以看出来，在这个例子里挑战和加油是关键词。

只要你不断重复这两个词，对方对你的好感度就会不断加深。而好感度越来越高，对方的心情就会越来越愉快，对你也会越来越信任，尤其是正处于恋爱阶段的朋友啊，用这个说话技巧更能打动爱人的心哦。第四种，引蛇触洞。

引蛇触洞是沟通读心书中的基本技巧之一，简单来说就是通过若无其事的问问题，从。🎼对方口中套出信息，还是举个例子给大家说明。张三问李四，你看起来心情不好，发生什么了。李四回答，没事儿，张三继续问。

工作上遇到问题了吗？还是股票跌了，难道是感情问题。李四打断张三说，关你什么事，你管好自己吧。张三说，我不就是想关心你一下吗？虽然是疏于好意的关心，但说谎的方式不对，就达不到好对方拉近距离的效果。

来看看使用引资触动这个技巧的效果。😊，🎼张三问李是，你看起来心情不好，是人际关系上出问题了吗？李是反问人际关系。张三继续说，对啊，像职场问题、朋友问题、感情问题这些了，李是回答道，哎，没错。

我和他是有点问题。张三说，难怪你心情不好，愿意和我说说吗？你是回答好，我跟你说说，你帮我分析分析。使用这个技巧，首先要选择一个宽泛的话题，像例子中的人际关系，这就具有很强烈的暗示性。

对方也很容易对号入座。然后在细节化话题引导对方，对方就会顺着你的话说出你想获取的信息。🎼不论是情场还是职场，顺立沟通的第一步，就是让对方信任你，而前提就是需要你先向对方敞开心扉。但开门见上的告诉对方。

我只要你信任，反倒会激起对方的防备。因此，必须通过暗示，让对方潜移默化的信任你才有用。而所谓沟通笃心术，就是在尊重对方了解对方认同对方的基础上，在通过一些启发性、询问性、暗示性的说话技巧。

能让人产生信任感，获取了信任感后，在好好经营这种信任关系，一定会大有收获。
