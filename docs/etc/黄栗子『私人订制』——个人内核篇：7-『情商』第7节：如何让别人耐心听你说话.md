# 黄栗子『私人订制』——个人内核篇：7-『情商』第7节：如何让别人耐心听你说话

🎼会聊天不仅是人们眼中的高情商，更是一种帮助你打破自身局限，在交流中展现自我并实现自我的技能。在医学上，有些药本身不起作用，而是为别的药服务的，他们就被称为药引子。而在人际交往过程中。

有些沟通啊也是为了让之后的交流顺通无阻，所以也可以被叫做是人际关系上的药引子。🎼每个人或多或少都遇到过这样的情况，那就是你想说话，但留给你说话的时间有限。比如客户只给你两分钟介绍产品。

领导没工夫听你聊项目规划，女朋友生气，不听你解释，遇到这样的情况，怎样才能让他们有耐心的听我们讲话呢？🎼可能很多朋友会想到啊，既然时间不等人，那就长话短说，废话不说。但往往真遇到这样的情况时。

人们就会恨不得自己是个相声演员，一口气就把事情给说完，先不谈你能不能把想说的话说完，可能就算你说完了，别人也理解不了你的意思。为什么会造成这样的局面呢？其实是因为我们忽略了最关键的一点。

沟通是双向的好的沟通并不是指你把事情一股老脚完了，任务就结束了。单方面的传达信息不去管听的人有没有听进去。这场沟通啊无疑是浪费时间和精力的。在和别人说话的时候，不仅要保证自己表达完善。

还要保证听的人能够投入，这样的沟通才是成功的。那么如何才能让别人耐心且认真的听你说话呢？换句话说，如何让别人给你时间去说话呢？时间是阻碍沟通的最大绊脚石，也是成就沟通最好的垫脚式。因此。

在和别人沟通的时候，我们最需要的，往往就是为自己赢取时间。这样才能让别人耐心且认真。🎼听我们说话。在这里例子先给大家分享一个小技巧，以小博大。怎么去以小博大呢？简单来说。

当遇到对方不给我们足够的时间去说话的时候，先别急着想，如何在短时间内说完自己要说的话，而是要先思考怎么样才能让别人愿意花更多的时间来听我们说话。比如去到一家公司面试，面试官对你说。

现在给你60秒展示一下你的特长，这种时候你会怎么做呢？60秒的时间，一首歌的时间都不够，能够做什么呢？又怎么能够展示特长呢？所以这个问题本身就有问题。那么他的答案就不会是根据字面意识得出的。

面试官的目的是想考验面试者在条件不充分的时候，是否有创造条件解决问题的能力。因此啊这个时候需要做的其实不是展示自己的特长，而是争取到能够展示特长的时间。🎼假如我们不具备能当场展示的才艺。

如唱歌、跳舞之类的那我们完全可以这么对面试官说，让我60秒内展示特长，这个挺困难的，特别是对我来说，其实在工作的时候，我是一个没什么魅力的人，存在感很弱。

可也不知道为什么身边的些朋友啊都说和我相处时很舒服，和我共识很安心，可能这是一种特长吧，所以可能会有点超时，但不知道我可以讲几个关于这方面的例子吗？🎼在这短短几句话中包含了三层含义。第一，制造意外。

面试官要求展示特长，但是你不按常理出牌，告诉面试官，你是一个没什么特长的人，至少在当下的场合无法展示出来。🎼第二，激发好奇。这段话中啊给了面试官一个疑问，既然没有魅力可言，那为什么别人喜欢和你待在一起。

喜欢和你共事呢？第三，回应主题，当面试官对你所表示的没有魅力，但受欢迎的这个特点感兴趣时，你就可以开始举几个例子，表达自己是如何默默无闻，但又人缘好的，哪怕最后时间超过60秒，面试官也不会计较。

因为我们已经成功争取到了更多的时间，其实以小博大这样的智慧，古人早已参透了其中的奥意。在春秋战国时期，齐国贵族田英打算在自己的封地修筑城墙。但这样做啊，无疑会引起齐王的猜疑。

于是田英府上的门客都来劝阻他，最后，田英实在是被劝的心烦了，直接下命令不再见前来劝说了门客。而有一个聪明的访客想了一招。他对田英说，臣子说三个字，多一个字，你就把我处使。这就是我们刚刚讲到的制造意外。

果然，田爷一听心想，三个字就想说服我，好，我倒要听听你想说什么。🎼于是这个访客啊通过以小博大的技巧，成功在一句话也听不进去的田英那里争取到了让他听三个字的机会。接下来就是激发好奇。

于是访客恭恭敬敬对田英说，海大鱼说完后转身就走，这下就成功激发了田英的好奇心，对于这三个字。田英是丈二和尚摸不着头了，连忙喊住访客，对他说，请等一等，留下把话说完吧。访客依旧不回答。

表示一开始说好只说三个字，不敢拿自己的生死当儿戏。田英只好反过来安慰访客让他淡送无仿，不会自罪于他。访客用海大鱼这三个字激发了田英的好奇心，成功将原本说三个字的机会，又进一步争取到了畅所欲言的时间。

而他完整的观点与想法，也是在之后争取到了这段时间里才开始表达最后就是回应主题。访客激发了田英的好奇心后，还是要回到主题上来。并且还得讲出一番道理。不然对方发现你只是故弄玄虚，也不会有一个。🎼好结果。

🎼访客争取到时间后，对田英说，海水里面的大鱼，渔网捕不到它，鱼钩也钓不到它，但要是在没有水的地方，连小小的蚂蚁也可以欺负他，而强大的齐国就像是大海，您就像是海中的大鱼。如果您有齐国的庇护。

那还修筑这城墙做什么呢？如果您失去了齐国的庇护，及时把城墙修筑了和天一样高，又有什么用呢？听完访客的话后，田英觉得很有道理，于是放弃了修筑城墙的打算，这就是一个运用以小博大的经典案例，通过制造意外。

激发好奇，回应主题，将原本小小的沟通机会扩大到让对方心甘情愿想要听下去的地步。因此，盗芒遇到表达时间很紧迫的时候，千万不要一根筋的按照原来的思路讲下去，也不要把话说的像连珠炮似的，让别人听着就厌烦。

而是要先想办法为自己争取时间，让对方有耐心且专心的去听。这样我们才有机会完整的说出自己的想法。🎼最后例子要提醒大家，在运用以小博大这个技巧时，要分清楚情况。这个技巧只能用于他人不给你时间说话的情况。

不能用于局势本身就十分紧张的情况，就好比一个地方失火了。你要做的事赶紧告诉消防队失火的位置在哪里，而不是去解释为什么会失火。还需要注意有胆量以小博大，你就得有底气，用好争取来的时间。

不然你成功引起了对方的注意，最后却讲不出个所以然，只会适得其反。
